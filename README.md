# chocolate-classifier
Simple chocolate classifier that is able to classify given image of a chocolate between 12 different available categories:

* Aero
* Coffee Crisp
* Hershey's
* Kit Kat
* Kinder
* Mars bar
* Reese's
* Snickers
* Skor
* Toblerone
* Twix
